Counter

Descripción: Este componente interactivo nos permite modificar un contador numerico a traves de varios botones, cada uno con una funcionalidad especifica:

"+": el contador se incrementa en 1.
"++": el contador se incrementa en 10.
"Reset": el contador se restablece a 0.
"Random": el contador se incrementa o decrementa por un número aleatorio entre -20 y 20.
"-": el contador se decrementa en 1.
"--": el contador se decrementa en 10.

Filter

Descripción: Este componente muestra una lista de tarjetas de Pokémon. El usuario puede filtrar la lista seleccionando un tipo de Pokémon en el selector en la parte superior. La lista se actualizará automáticamente para mostrar solo los Pokémon que tienen el tipo seleccionado.'

Shooping Cart

Descripción: Este componente muestra una lista de productos más el carrito de compras o shopping cart. Cuando el usuario aún no ha elegido ningún producto, el carrito muestra el mensaje de "No products", una vez el usuario empieza a agregar productos; se mostrará un resumen (nombre de producto, cantidad y subtotal) más el monto total.
